🐍 Sharing my 2nd semester Python project!

Automated Resume Generator — a Flask web app that 
takes your details and instantly generates a 
professionally designed PDF resume.

Built using:
→ Python + Flask (REST API)
→ fpdf2 (PDF generation)
→ Object-oriented programming
→ JSON data handling

The app generates a real two-column PDF with sidebar, 
avatar initials, skills, experience and education 
sections — all from a single POST request. 🚀

💻 GitHub: [https://github.com/AbithaMuniyan/Automated-resume-generator/blob/main/resume-generator-README.md]
